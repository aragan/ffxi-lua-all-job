Nasomi Notes

HQ% Chance by tier (from DarkStar code): 
    0-10 skill  = T0 = .015 = 1.5% hq
    11-30 skill = T1 = .1   = 10% hq
    31-50 skill = T2 = .3   = 30% hq
    51+ skill   = T3 = .5   = 50% hq
HQ% Chance mods (from DarkStar code): 
    //new moon +33% of base rate bonus to hq chance
    //full moon -33%
    //corresponding/weakday/lightsday -33%
    //opposing/darksday +33%
    if(chance > 0)

    else if (WeekDay == LIGHTSDAY)
